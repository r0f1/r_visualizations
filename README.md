# Visualizations with R

Visualizations with R and ggplot.

Go to [https://r0f1.github.io/](https://r0f1.github.io/) to view the project.

## Installation

* Install [git lfs](https://git-lfs.com/).
* Install [just](https://github.com/casey/just).
* Clone the repository.
* Download [fontawesome](https://fontawesome.com/download) and extract it in the top-level folder.
* Run `just init`.
